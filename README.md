# PD Auto Flow VIP — Update Server

ไฟล์นี้ใช้สำหรับเช็คเวอร์ชันอัพเดตของ PD Auto Flow VIP Extension

## Files
- `version.json` — ข้อมูลเวอร์ชันล่าสุด (บอทเช็คจากไฟล์นี้)
- Releases — ไฟล์ ZIP สำหรับดาวน์โหลด

## วิธีอัพเดตเวอร์ชัน
1. แก้ `version.json` → เปลี่ยน version + changelog
2. อัพโหลด ZIP ใหม่ใน GitHub Releases
3. บอทลูกค้าจะแจ้งเตือนอัตโนมัติ
